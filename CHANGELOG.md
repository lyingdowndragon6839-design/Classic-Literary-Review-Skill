# Changelog

All notable changes to this project are documented here.

## v2.0.0 - 2026-06-27

### Changed

- Standardized repository structure for long-term GitHub maintenance.
- Renamed project files to lowercase kebab-case where appropriate.
- Moved internal writing system files into `modules/`.
- Rebuilt README with quick start, table of contents, examples, roadmap, contributing, license, and contact sections.
- Added GitHub standard files: `LICENSE`, `CONTRIBUTING.md`, `.gitignore`.
- Added module and example indexes.

### Fixed

- Updated internal links after file renames.
- Reorganized examples so each book has a consistent standalone file.
- Cleaned Markdown headings, tables, code fences, and checklist formatting.

## v1.1.0 - 2026-06-27

### Added

- Added research, style learning, writing, revision, humanization, AI check, and reviewer modules.
- Added usage examples and full workflow examples for several classic works.
- Added offline mode, research ethics, source log schema, and output contracts.

## v1.0.0 - 2026-06-27

### Added

- Initial Markdown-first Skill repository.
- Added core writing workflow for Chinese reading responses and literary reviews.
- Added first batch of work examples and reusable templates.
