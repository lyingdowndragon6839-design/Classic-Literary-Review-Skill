# Search

搜索流程的快捷入口。完整执行规则见 [modules/research.md](modules/research.md)。

## 什么时候搜索

- 用户允许联网
- 作品已有大量公共评论，容易落入俗套
- 用户需要课堂提交、平台长评或正式文章
- 需要确认常见观点、少见角度和可用细节

## 输出要求

搜索后至少给出：

| 项目 | 说明 |
|---|---|
| Source List | 已读来源、链接、访问状态 |
| 高频观点 | 已经被反复写过的角度 |
| 少见角度 | 更值得展开的判断 |
| 写法可借鉴处 | 开头、转场、结尾、节奏 |
| 避免使用 | 套话、复制风险、俗套角度 |

离线时按 [docs/offline-mode.md](docs/offline-mode.md) 执行。
