# AI Check

AI trace is not only a word problem. A paragraph can avoid every forbidden phrase and still feel
generated if the logic is too smooth, the rhythm too even, or the ending too obedient.

## Reading Response Cliche Table

| Cliche | Better Direction |
|---|---|
| 这本书让我受益匪浅 | 我合上书以后，最先想到的不是“收获”，而是一种说不清的沉重。 |
| 引人深思 | 这一幕真正留住人的，是它没有把问题说完。 |
| 发人深省 | 读到这里，人很难再轻易下判断。 |
| 体现了人性的光辉 | 这个选择动人，是因为他明明可以少承担一点，却没有退开。 |
| 揭示了社会的黑暗 | 黑暗不是背景，而是工资、身份、名声和机会一点点压到人身上。 |
| 反映了时代的悲剧 | 悲剧不是时代两个字造成的，而是那个时代让人只剩下很窄的选择。 |
| 命运的无常 | 生活刚给他一点落脚处，下一页就把那点落脚处撤走。 |
| 苦难中的希望 | 希望在这里并不明亮，它只是人在撑不下去时还没松手。 |
| 生命的意义 | 这本书没有急着回答意义，它只是逼人看见一个人为什么还继续活着。 |
| 让我明白了 | 我读到最后才意识到，前面那些细节一直在把人推向这个结局。 |
| 值得我们每个人思考 | 这个问题并不宏大，却很难躲开：换成我们，会不会也做出相似选择？ |
| 总而言之 | 到这里，不需要再总结；那个场景本身已经把话说完了。 |
| 由此可见 | 也正因为这个细节，前面对人物的判断才突然变得不那么简单。 |
| 深化了主题 | 它让问题从“发生了什么”变成“人为什么只能这样选择”。 |
| 升华了主旨 | 结尾没有把主题抬高，而是把读者重新按回那个具体的人身上。 |

## Logic Check

| Problem | Symptom | Repair |
|---|---|---|
| 观点跳跃 | From a scene directly to “人性/时代/命运” | Add the missing reasoning step |
| 论据不足 | Strong claim, weak evidence | Add a scene, action, object, or narrator choice |
| 剧情和观点脱节 | Plot is accurate but does not support the thesis | Cut or reconnect to the main judgment |
| 前后矛盾 | Early paragraph praises what later paragraph rejects | Name the change or revise one side |
| 空泛结尾 | Ending gives advice to everyone | Return to the book's final pressure point |
| 为深刻而深刻 | Sentence sounds impressive but cannot be tested | Replace with a smaller, sharper claim |

## Rhythm Check

- Paragraphs should not all have the same length.
- Not every paragraph needs a summary sentence.
- Questions should open thought, not decorate.
- A short sentence should land after a real turn, not appear randomly.
- Personal reading moments should be few but specific.
