# 用户使用流程示例

这个文件展示普通用户怎样一步步使用 Classic Literary Review Agent 写读后感。

## 示例一：《悲惨世界》完整使用流程

### 用户怎么输入

```text
我想写《悲惨世界》的读后感，1500 字，课堂提交版。请先联网搜索高质量读后感和文学评论，提炼常见观点和容易写俗的地方，再帮我选一个最有力量的主判断，最后写成自然、不像 AI 的文章。
```

### Skill 应该先做什么

1. 搜索知乎、豆瓣、微信公众号、文学评论网站等来源。
2. 总结常见角度：苦难、善良、救赎、法律、社会黑暗。
3. 标出俗套风险：只写“善良战胜苦难”，容易变成空泛赞美。
4. 提炼可写角度：
   - 冉阿让如何从“犯人”重新成为人。
   - 沙威为什么不是普通反派。
   - 慈悲为什么不是温柔，而是改写身份的行动。

### Skill 不应该直接做什么

不应该一上来写：

```text
《悲惨世界》是法国作家雨果的代表作，它揭示了社会的黑暗，体现了人性的光辉……
```

这类开头太像教材和 AI 总结。

### 如何提炼观点

推荐主判断：

```text
《悲惨世界》真正打动人的，不是苦难本身，而是一个人能不能从社会已经写好的名字里重新长出来。
```

为什么这个观点好：

- 能统领冉阿让、沙威和芳汀。
- 不只是赞美善良。
- 有具体人物和情节支撑。

### 如何写初稿

初稿结构：

| 部分 | 内容 |
|---|---|
| 开头 | 避开“世界名著”介绍，从“犯人这个名字”进入 |
| 中段一 | 主教如何替冉阿让改写身份 |
| 中段二 | 沙威为什么无法接受人的改变 |
| 中段三 | 芳汀说明贫困怎样毁掉人的名字 |
| 结尾 | 回到“人是否只能等于过去的标签” |

### 如何修改

修改重点：

- 删除“体现人性光辉”。
- 减少剧情概括。
- 增加“我读到沙威崩溃时才意识到……”这种阅读过程。
- 把结尾从道德口号改成具体问题。

### 如何降低 AI 味

把：

```text
这本书发人深省，让我们明白善良具有重要意义。
```

改成：

```text
读到最后，我反而不太敢轻易说“善良”这个词。因为在这本书里，善良不是一句好听的话，而是一个人一次次冒着被重新拖回旧身份的风险，仍然不肯把别人交给命运。
```

## 示例二：《生死疲劳》完整使用流程

### 用户怎么输入

```text
我刚读完《生死疲劳》，想写一篇 1200 字读后感，但现在只有一些零散感受：轮回、土地、苦难、荒诞、历史压迫感。请先不要直接写文章，先帮我整理这些感受，提炼 3 个可写主判断。
```

### Skill 应该先做什么

1. 先整理用户感受，不急着写。
2. 搜索相关评论，特别关注“轮回结构”“土地叙事”“魔幻现实主义”“乡村历史”。
3. 区分常见观点和可写新角度。

### Skill 不应该直接做什么

不应该直接写成：

```text
《生死疲劳》通过荒诞的手法反映了时代的悲剧，揭示了人性的复杂。
```

这种说法太空，几乎可以套给很多小说。

### 如何提炼观点

可选主判断：

1. 轮回不是叙事花招，而是看历史的装置。
2. 动物视角让人类的庄严叙事露出粗糙边缘。
3. 《生死疲劳》的荒诞不是逃离现实，而是现实本身已经荒诞到需要变形。

最推荐：

```text
《生死疲劳》用轮回写历史，不是为了制造奇观，而是为了让历史的荒唐无法被正常叙事消化。
```

### 如何写初稿

结构：

| 部分 | 内容 |
|---|---|
| 开头 | 从“轮回不是热闹”进入 |
| 中段一 | 西门闹为什么换身体仍被土地牵住 |
| 中段二 | 动物视角怎样拆开人的庄严说法 |
| 中段三 | 历史压力怎样进入乡村日常 |
| 结尾 | 回到“历史换一种身体回来” |

### 如何修改

- 减少“魔幻现实主义”的概念解释。
- 增加土地、身体、动物视角等具体抓手。
- 删掉“时代悲剧”“人性复杂”。

### 如何降低 AI 味

把：

```text
作品揭示了历史的沉重和生命的意义。
```

改成：

```text
我读到后面才觉得，轮回并不是让人逃离历史。相反，它像是历史不肯离场，只能换一种身体、换一种叫声、换一种更别扭的方式回来。
```

## 示例三：《活着》完整使用流程

### 用户怎么输入

```text
我要交一篇课堂读后感，作品是《活着》，字数 1000 字。读完后最难受的是福贵最后只剩下老牛作伴。请围绕这个感受帮我提炼主判断，搭结构，写初稿，再把文章改得不励志、不像 AI。
```

### Skill 应该先做什么

1. 保留用户“老牛作伴”这个真实感受。
2. 搜索《活着》高质量读后感和评论。
3. 标出俗套风险：不要写成“苦难使人坚强”。
4. 选择更具体的主判断。

### Skill 不应该直接做什么

不应该写：

```text
《活着》让我受益匪浅，它告诉我们无论生活多么艰难，都要坚强地活下去。
```

这会把余华的冷写成励志口号。

### 如何提炼观点

推荐主判断：

```text
《活着》最残酷的地方，不是苦难把人变伟大，而是苦难把人一点点磨空以后，人仍然还在活着。
```

### 如何写初稿

结构：

| 部分 | 内容 |
|---|---|
| 开头 | 反对“苦难使人坚强” |
| 中段一 | 福贵不是英雄，他只是不断失去 |
| 中段二 | 余华为什么写得那么平静 |
| 中段三 | 老牛结尾为什么沉重 |
| 结尾 | 不回答生命意义，只留下继续活着 |

### 如何修改

- 删除“受益匪浅”“生命意义”“坚强”等词。
- 保留课堂提交需要的清楚结构。
- 加入用户读到老牛结尾时的真实反应。

### 如何降低 AI 味

把：

```text
这本书让我明白了生命的意义。
```

改成：

```text
读到福贵和老牛那一段，我很难把它说成“生命的意义”。它更像是意义已经不太说得出口了，可人还是没有停下。
```
