# 《悲惨世界》Agentic Example

## 作品简介

雨果以冉阿让、芳汀、珂赛特、沙威和街垒青年为核心，把个人救赎、法律秩序、
  贫困结构和宗教慈悲放在同一部巨大的小说里。

## 主判断

《悲惨世界》真正有力的地方，不是说善良可以战胜苦难，
  而是说一个人能不能从制度已经写好的名字里重新长出来。

## 写作思路

- 刺痛点：冉阿让总是被“犯人”这个旧名字追上。
- 矛盾点：沙威并不庸俗，他只是完全相信秩序。
- 忽略点：慈悲不是温柔情绪，而是改写身份的行动。
- 反直觉点：法律越干净，越可能看不见具体的人。

## 文章框架

| 段落 | 功能 | 证据 |
|---|---|---|
| 1 | 修正常见误读 | 不从“苦难中的善良”起笔 |
| 2 | 分析主教一幕 | 银器、烛台、身份重命名 |
| 3 | 写沙威 | 秩序信仰与崩溃 |
| 4 | 写芳汀 | 贫困如何进入身体与名声 |
| 5 | 收束 | 人是否只能等于档案里的名字 |

## 开头示例

很多人写《悲惨世界》，喜欢从“善良战胜苦难”开始。这个说法太顺了，
  顺到几乎遮住了小说真正尖锐的地方：雨果一直在追问，一个人被社会命名之后，
  还有没有机会重新成为另一个人。

## 正文示例

主教放走冉阿让，不只是宽恕了一个偷银器的人。他做的是更危险的事：
  他当着所有人的面替冉阿让改写身份。法律说你是犯人，主教说你是被我托付的人。
  这个动作不是抒情，它几乎是在公开挑战一套分类系统。

## 结尾示例

所以这本书留下的不是“我们要善良”这样轻的道理。它更像是在问：
  当一个人的过去被写进档案、写进目光、写进所有人的称呼里，他还要付出多少代价，
  才能重新拥有一个不是惩罚的名字？

## 为什么这样写

这样写避开了剧情复述，也避免把冉阿让写成单纯励志人物。文章把慈悲、法律、
  贫困和身份放在同一条线上，主判断更集中。

## 修改过程

| 轮次 | 修改动作 | 结果 |
|---|---|---|
| v1 | 删除“人性的光辉” | 改为“改写身份” |
| v2 | 压缩剧情 | 情节只保留主教、沙威、芳汀 |
| v3 | 增加矛盾 | 沙威不再是脸谱反派 |
| v4 | 调整结尾 | 从口号改为具体问题 |

## 最终版本

《悲惨世界》并不只是讲苦难里仍然有善良。这个说法当然对，但它太轻，
  轻到几乎把雨果真正要处理的问题遮住了。冉阿让最难摆脱的，不是饥饿，不是贫穷，
  甚至不是追捕，而是那个已经贴在他身上的名字：犯人。

主教放走他的一幕，因此不只是温情。主教没有说“我原谅你”就结束，
  他当着警察的面把银器变成赠礼，把偷窃变成托付，
 把一个旧身份硬生生改写成新的可能。 慈悲在这里不是情绪，是行动，是冒险，
 是让制度的判断短暂失效。

沙威的存在让这个问题更锋利。他不是普通恶人。他可怕，正因为他干净、忠诚、
  没有私心。他相信法律记录比人更可靠，相信一个人的过去可以决定他的全部。
  冉阿让每一次成为更好的人，都在破坏沙威的世界。

所以小说真正刺人的地方，是它不让“善良”停在赞美词里。芳汀的坠落告诉我们，
   贫困会进入工资、名声、身体和母亲的身份；冉阿让的逃亡告诉我们，
  一个人即使改变了， 也要不断证明自己不是过去那个名字。

读完这本书，最难放下的不是苦难有多深，而是人要怎样从别人已经写好的命运里，
   把自己重新夺回来。
