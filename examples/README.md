## Example Standard

- 只保留足够支撑写作判断的剧情。
- 每个示例必须能说明一个具体写法。
- 最终版本不能像通用范文。
- 新增示例请使用 [../assets/templates/example-case.md](../assets/templates/example-case.md)。
