# Style Learning

风格学习的快捷入口。完整执行手册见 [modules/style-learning.md](modules/style-learning.md)。

## 原则

学习写法，不学习句子。

风格学习只记录文章怎样开头、怎样转弯、怎样从细节进入判断、怎样收束，
 不保存可识别的原句或独特比喻。

## 建议产物

| Bank | 记录什么 |
|---|---|
| Opening Bank | 好文章如何避开作者生平和剧情概括 |
| Transition Bank | 怎样从情节转到判断 |
| Ending Bank | 怎样不靠口号收尾 |
| Question Bank | 哪些问题能继续生出分析 |
| Rhythm Bank | 段落长短、停顿、短句位置 |
| Image Bank | 哪些物件或场景承载意义 |
