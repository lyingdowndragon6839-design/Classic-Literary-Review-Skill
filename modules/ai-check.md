# AI Check

AI 痕迹不只是词的问题。一段话可以避开所有禁用词，但如果逻辑太顺、 节奏太均匀、
结尾太乖巧，仍然读起来像生成的。

## 读后感套话对照表

| 套话 | 更好的方向 |
|---|---|
| 这本书让我受益匪浅 | 我合上书以后，最先想到的不是"收获"，而是一种说不清的沉重 |
| 引人深思 | 这一幕真正留住人的，是它没有把问题说完 |
| 发人深省 | 读到这里，人很难再轻易下判断 |
| 体现了人性的光辉 | 这个选择动人，是因为他明明可以少承担一点，却没有退开 |
| 揭示了社会的黑暗 | 黑暗不是背景，而是工资、身份、名声和机会一点点压到人身上 |
| 反映了时代的悲剧 | 悲剧不是时代两个字造成的，而是那个时代让人只剩下很窄的选择 |
| 命运的无常 | 生活刚给他一点落脚处，下一页就把那点落脚处撤走 |
| 苦难中的希望 | 希望在这里并不明亮，它只是人在撑不下去时还没松手 |
| 生命的意义 | 这本书没有急着回答意义，它只是逼人看见一个人为什么还继续活着 |
| 让我明白了 | 我读到最后才意识到，前面那些细节一直在把人推向这个结局 |
| 值得我们每个人思考 | 这个问题并不宏大，却很难躲开：换成我们，会不会也做出相似选择？ |
| 总而言之 | 到这里，不需要再总结；那个场景本身已经把话说完了 |
| 由此可见 | 也正因为这个细节，前面对人物的判断才突然变得不那么简单 |
| 深化了主题 | 它让问题从"发生了什么"变成"人为什么只能这样选择" |
| 升华了主旨 | 结尾没有把主题抬高，而是把读者重新按回那个具体的人身上 |

## 逻辑检查

| 问题 | 症状 | 修复 |
|---|---|---|
| 观点跳跃 | 从场景直接跳到"人性/时代/命运" | 补上缺失的推理步骤 |
| 论据不足 | 强烈主张，薄弱证据 | 加入场景、动作、物件或叙述者选择 |
| 剧情与观点脱节 | 剧情准确但不支撑判断 | 删除或重新连接到主判断 |
| 前后矛盾 | 前段赞美的后段推翻 | 命名这个变化，或修正其中一边 |
| 空泛结尾 | 结尾给所有人提建议 | 回到书里最后一个压力点 |
| 为深刻而深刻 | 句子听起来厉害但无法检验 | 换成更小、更锋利的判断 |

## 节奏检查

- 段落不应该全一样长
- 不是每段都需要总结句
- 疑问应该打开思考，不是装饰
- 短句应该落在真正的转折之后，不是随机出现
- 个人阅读时刻应少而具体
