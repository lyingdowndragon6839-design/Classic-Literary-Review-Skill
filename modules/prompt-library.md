# Prompt Library

可复制的用户提示词。这些提示词之所以有效，是因为它们要求过程先于成稿。

## 读后感（通用）

```text
我想写《活着》的读后感，1000 字，课堂提交版。我读完后最难受的是福贵最后只剩下
老牛作伴。请先帮我提炼 3 个可写主判断，选一个最有力量的角度，再搭结构，最后写
初稿并降低 AI 味。
```

**为什么有效**：给出了作品、字数、真实阅读感受、提交场景，并要求判断先于成稿。

## 文学评论

```text
我想写一篇《罪与罚》的文学评论，不要写成普通读后感。请先联网搜索高质量评论，
比较常见的"罪与救赎"角度和更少见的"理论与身体冲突"角度，再帮我确定一个可论证
的主判断和文章结构。
```

**为什么有效**：要求 Agent 比较角度再选择，而不是直接涌向最熟的角度。

## 知乎回答

```text
我想把《百年孤独》写成知乎长回答风格。请先搜索知乎高赞回答常见结构，学习它们
如何开头、转折和收束，但不要复制句子。文章需要有明确判断和个人阅读体验。
```

**为什么有效**：指定平台风格，同时保护原创性。

## 课堂读后感

```text
我要交一篇课堂读后感，作品是《悲惨世界》，1500 字。要求观点清楚，有文本细节，
不要太散文，也不要太论文。请先帮我找一个不俗套的主判断，再搭结构。
```

**为什么有效**：控制语气的分寸——不太散、不太硬。

## 公众号文章

```text
我想写一篇公众号风格的《老人与海》读后感。请不要写成励志文。先帮我搜索相关评论，
找出常见俗套，再围绕"尊严不在胜利，而在失败后行动仍有重量"这个角度写一篇自然、
有节奏的文章。
```

**为什么有效**：明确说出了想避免的俗套，同时给出了更锋利的角度。

## 演讲稿

```text
我需要把《平凡的世界》读后感改成 3 分钟演讲稿。请保留一个主判断，减少复杂论证，
增加听众能跟上的转场和停顿，不要写成朗诵腔。
```

**为什么有效**：口语写作需要节奏，不是密集分析。

## 只降 AI 味

```text
下面是我的草稿。请只做降 AI 味修改，不要换掉我的观点。重点检查套话、空泛总结、
段落过于整齐、剧情和观点脱节、结尾像万能作文的问题。
```

**为什么有效**：明确告诉 Agent 做编辑，不做重写。
