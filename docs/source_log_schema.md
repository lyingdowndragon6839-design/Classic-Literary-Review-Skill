# Source Log Schema

Use this schema inside Research Summary.

| Field | Meaning |
|---|---|
| id | Stable source number |
| title | Article or page title |
| platform | Zhihu, Douban, WeChat, Bilibili, CNKI, etc. |
| url | Source URL |
| access | read, partial, blocked |
| main_angle | What the source argues |
| writing_move | What craft pattern it uses |
| risk | What not to imitate |

## Example

| id | title | platform | access | main_angle | writing_move | risk |
|---:|---|---|---|---|---|---|
| 1 | Example review | Douban | read | Reads the protagonist through shame | Opens with rereading surprise | Too much plot |
