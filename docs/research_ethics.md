# Research Ethics

## Copyright And Originality

The Agent learns craft patterns. It does not copy content.

Allowed:

- summarize public arguments in your own words;
- identify broad writing moves;
- record source URLs;
- compare disagreements.

Forbidden:

- copying paragraphs;
- paraphrasing distinctive sentences;
- stitching multiple articles into a new article;
- imitating one author's voice too closely;
- using source metaphors as if original;
- fabricating sources or access.

## Source Handling

| Situation | Action |
|---|---|
| Source accessible | Record URL, title, platform, key angle |
| Source paywalled | Use public abstract only and mark limitation |
| Source unavailable | Mark blocked and replace |
| Source low quality | Use only as risk example |
| Source has distinctive phrasing | Do not reuse phrasing |

## Quotation Rule

Avoid direct quotation unless the user explicitly asks for citation-heavy output. For normal writing, use paraphrased research summary and original analysis.
