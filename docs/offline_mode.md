# Offline Mode

## Activation

Offline mode is allowed only when:

1. internet access is unavailable; or
2. the user explicitly asks not to browse; or
3. the user provides all source material and asks the Agent to rely only on it.

## Required Disclosure

Use this note:

> 当前按离线模式执行：我没有联网检索，因此不会声称阅读过外部文章。下面的研究判断只基于用户提供材料与模型已有知识，可靠性低于联网模式。

## Offline Substitutes

| Online Step | Offline Substitute |
|---|---|
| Search 20+ pieces | Extract 20+ notes from user materials or known text details |
| Research Summary | Offline Reading Summary |
| Prompt Design Summary | Use repository workflow only |
| Style Learning | Build banks from user-provided examples |
| Reviewer | Still required |

## Restrictions

- Do not invent source URLs.
- Do not say “检索发现”.
- Do not present current online consensus.
- Do not claim high confidence on obscure facts.
