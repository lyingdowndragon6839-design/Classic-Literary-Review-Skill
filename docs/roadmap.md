# Roadmap

这个仓库按长期维护项目来整理，不按一次性 Prompt 来维护。

## 已完成

| 模块 | 状态 |
|---|---|
| Research Workflow | 已完成 |
| Style Learning | 已完成 |
| 读后感专用流程 | 已完成 |
| Reviewer Agent | 已完成 |
| AI Check | 已完成 |
| Prompt Library | 已完成 |
| Decision Tree | 已完成 |
| 完整流程示例 | 已完成 |

## 下一步

- 增加更多完整流程示例，覆盖中国、俄国、法国、拉美、英语文学。
- 增加更多写作场景：课堂读后感、豆瓣长评、知乎回答、公众号文章、演讲稿。
- 增加教师评分表和同伴互评表。
- 增加带真实搜索记录的 Source Log 示例。
- 增加脚本化 Markdown 健康检查。

## 未来

- MCP 支持：自动保存搜索来源和访问状态。
- 真正多 Agent 执行：研究、构思、审稿、改稿分工。
- 本地读书笔记导入。
- 根据失败草稿自动优化 Prompt。
- 从用户认可的文章中持续积累个人 Style Bank。

## 维护原则

每次新增内容都要问一句：它是否真的帮助用户写出更好的读后感？如果只是显得高级，
但不能指导写作，就删掉。
